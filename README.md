# Superstore Sales Analysis

## Overview
Analysis of 9,994 sales records to identify revenue trends, 
top products, and regional performance.

## Tools Used
- Python (pandas) — data cleaning
- MySQL — data analysis and querying
- Tableau — interactive dashboard

## Key Findings
- West region generated the highest total sales
- Tables sub-category had negative profit despite high sales volume
- Q4 consistently outperformed other quarters
