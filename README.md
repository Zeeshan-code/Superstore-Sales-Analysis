# Superstore Sales Analysis

## Overview
Analysis of 9,994 sales records to identify revenue trends, 
top products, and regional performance.

## Tools Used
- Python (pandas) — data cleaning
- MySQL — data analysis and querying
- Tableau — interactive dashboard

## Key Findings
- West region led all regions with $725K in sales and $108K in profit
- Tables sub-category lost $17,725 despite $206K in sales — highest loss of any category
- Sales grew 51% over 4 years, from $484K in 2014 to $733K in 2017
- Canon imageCLASS 2200 Copier was the single most profitable product
- Copiers had the best profit margin despite not being the highest revenue category
